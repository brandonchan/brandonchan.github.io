#map

BUT WHAT ABOUT EACH? I've decided to cover #map since I see so many examples that have used map. The most powerful function of #map, is that it manipulates the array or hash it is being used on. In the case with #each, it cannot manipulate the array. #map requires only one line and it's defined block. #each in the same steps cannot change the original array and requires a new array to house the changes. #map is a method with it's intention to manipulate the array or hash. The benefit to map as it doesn't change the original array, it sort of creates a temporary copy of the hash or array, and you can put it into a new array or keep the changes as the same array. But in my research of map, when you need to manipulate an array or hash, keep #map in mind.
